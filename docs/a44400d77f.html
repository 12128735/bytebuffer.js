Class: BufferView

BufferView

An optimized DataView-compatible BufferView for node Buffers without assertions.

new BufferView(buffer, byteOffset, byteLength)

Constructs a new BufferView.

Parameters:
Name Type Argument Description
buffer Buffer

An existing Buffer to use as the storage for the new BufferView object.
byteOffset number <optional>

The offset, in bytes, to the first byte in the specified buffer for the new view to reference. If not specified, the view of the buffer will start with the first byte.
byteLength number <optional>

The number of elements in the byte array. If unspecified, length of the view will match the buffer's length.
